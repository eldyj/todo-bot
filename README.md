# todo-bot
---
simple bot for one people oriented notes, suggestions, etc
</br>
## Preparing bot
as first you need to create `token.txt`
and put bot token here,
then just make `todos.json` with empty array `[]` inside
## Persimmions
all people can:
* see all todos
* see states of todos
</br>

owner can:
* change states of todos
* remove todos
* reset todos
